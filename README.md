# InterestCalculator

### Task
> build a simple interest calculator that, given a principal, rate of annual interest, and number of years, will display the total principal plus interest.

### Frameworks
  - React
  - Redux
  - Express.js

### Module Bundler
  - Webpack

### Transcompiler
  - Babel

### Run Interest Calculator Web App
  1. npm install
  2. npm run build
  3. npm start
